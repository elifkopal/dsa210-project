# Restaurant Success Analysis in Istanbul

## Overview
This project analyzes restaurant success in Istanbul by examining the effects of district and cuisine type.

## Data
Data was collected from Google Maps and includes:
- District
- Cuisine type
- Rating
- Number of reviews
- Price level

## Methods
- Data cleaning
- Exploratory Data Analysis (EDA)
- Hypothesis testing (ANOVA)
- Regression analysis

## Key Findings
- District does not have a statistically significant effect on success
- Cuisine type is a key determinant of success

Open the notebook and run all cells.
